◈ SCANNER MODULES
💉
SQL INJECTION
Tests form inputs and URL parameters with 10+ payloads. Detects error-based, blind, and UNION-based injection vectors by analysing database error signatures.
SEVERITY: CRITICAL
⚡
XSS DETECTION
Injects 8 reflected XSS payloads into every form field and checks whether the payload appears unsanitised in the HTTP response body.
SEVERITY: HIGH
🔒
SECURITY HEADERS
Audits 7 critical HTTP security headers including HSTS, CSP, X-Frame-Options, X-Content-Type-Options, and Referrer-Policy.
SEVERITY: MEDIUM
🔐
SSL/TLS ANALYSIS
Verifies certificate validity, expiry date, HTTPS enforcement, and flags certificate verification failures or impending expirations.
SEVERITY: HIGH
↗️
OPEN REDIRECT
Identifies redirect, url, next, return, and goto parameters across all crawled URLs and tests with external redirect payloads.
SEVERITY: MEDIUM
📂
FILE EXPOSURE
Probes 18 common sensitive file paths including .env, .git/config, wp-config.php, phpinfo.php, and backup files for accidental exposure.
SEVERITY: HIGH
